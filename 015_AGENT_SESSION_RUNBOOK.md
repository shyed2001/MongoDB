# Agent Session Runbook

## Start

1. Identify repo.
2. Read local instructions.
3. Read central index.
4. Read related registry card.
5. State intended actions.
6. Check risk gates.

## During Work

- Search before create.
- Prefer append-only edits.
- Keep logs.
- Use small patches.

## End

- Summarize changed files.
- Summarize risks.
- Summarize next actions.
- Update session log if needed.
