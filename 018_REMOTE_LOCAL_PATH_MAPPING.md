# Remote and Local Path Mapping

## Purpose

Map Windows local paths to repo-relative paths and remote Git URLs.

## Rule

Every important file should have:

- Windows local path
- repo-relative path
- remote URL
- Obsidian link if applicable

## Example

| Meaning | Value |
|---|---|
| Windows local path | `F:\GitHubDesktop\GitHubCloneFiles\AI_Tools_Plan_Master_Index_Lib\000_MASTER_INDEX.md` |
| Repo-relative path | `000_MASTER_INDEX.md` |
| Remote URL | `https://github.com/OWNER/AI_Tools_Plan_Master_Index_Lib/blob/main/000_MASTER_INDEX.md` |
| Obsidian link | `[[000_MASTER_INDEX]]` |
