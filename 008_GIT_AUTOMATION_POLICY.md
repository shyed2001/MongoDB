# Git Automation Policy

## Safe Commands

- git status
- git diff
- git log
- git branch --show-current
- git remote -v
- git fetch

## Ask First

- git add
- git commit
- git pull --rebase
- git push
- git reset
- git clean
- branch deletion
- tag deletion
- force push
- remote mutation

## Rule

Git automation must be loud, logged, reversible, and non-destructive by default.
