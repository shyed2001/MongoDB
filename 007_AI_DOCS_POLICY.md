# AI Docs Policy

## Rule

AI docs must be compact, linked, and cross-referenced.

## Do

- Use central index for global rules.
- Use local repo files for local exceptions.
- Use crossrefs for graph-style linking.
- Use templates for repeatable cards.

## Do Not

- Duplicate full master policy everywhere.
- Create many overlapping files without cross-links.
- Hide decisions in chat only.
