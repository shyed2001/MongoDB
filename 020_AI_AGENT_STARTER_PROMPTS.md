# AI Agent Starter Prompts

## Safe Repo Analysis

Read local repo instructions first, then read the central AI tools master index. Identify repo type, package manager, branch, remotes, protected files, and risk gates. Do not edit yet. Return a safe action plan.

## Safe Patch

Make only small, reversible, append-only edits where possible. Show changed files, risk level, and rollback method.

## Git Sync

Check status, branch, remotes, and local changes. Do not stage, commit, pull, rebase, or push without permission.
